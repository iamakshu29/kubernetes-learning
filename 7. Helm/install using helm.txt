You need to run to commands to install a package or application using helm.
1. First, add the Helm repository for the package you want to install. For example, to add the stable repository, run:
   helm repo add stable https://charts.helm.sh/stable



2. Next, install the package using the `helm install` command. For example, to install the nginx-ingress package from the stable repository, run:
   helm install <pod_name> repo/package_name

>> helm repo add bitnami https://charts.bitnami.com/bitnami
>> helm install my-nginx bitnami/nginx

To get all package in the repo.
>> helm search repo bitnami 

To search specific package in the repo.
>> helm search repo bitnami | grep nginx
